Namespace: Gateway

H. Gateway

Module to dialog between the apps of the Creative Suite.
This module allow you to execute a function in Photoshop from InDesign
Author:
  • Bastien Eichenberger

Methods

<static> call_app(app, function_to_execute, args, timeout) → {*}

Function to execute a script in an other software of the Creative Suite
Parameters:
Name Type Description
app String the name of the application [photoshop, illustrator, indesign, … ]
function_to_execute function the function to process
args Array an array of arguments
timeout Number the timeout in millisecond
Throws:
if an error occurred in the function passed as parameter or if the time exceeded without result
Type
Error
Returns:
the value returned by the function passed as parameter or null
Type
*
Example
With parameters
H.Gateway.call_app(
     'photoshop',
     function(a, b) {
         alert(a + b);
     },
     ['hello', 'world']
);

Without parameters
H.Gateway.call_app(
     'photoshop',
     function() {
         alert('function without parameter');
     }
);
